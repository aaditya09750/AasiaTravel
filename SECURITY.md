# Security Policy

## Supported Versions

We support security updates only on the active development branch of the Aasia Travel app.

| Version | Supported |
| ------- | --------- |
| v0.1.x  | Yes       |
| < v0.1  | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately.

Do **NOT** open a public issue. Instead, email reports to:
**aasiatravel@gmail.com**

We will acknowledge your report within 48 hours and coordinate a public release timeline for any fixes.
